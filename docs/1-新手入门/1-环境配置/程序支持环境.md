## 硬件环境

### 入门级

CPU内存1核1G、硬盘40GB、带宽1M

### 普通级

CPU内存1核2G、硬盘40GB、带宽2M

### 增强级

CPU内存2核4G、硬盘40GB、带宽4M

### 数据级

CPU内存4核16G、硬盘100GB、带宽4M

## 操作系统

### 推荐环境：

Linux CentOS 7.0 x64以上

### 其他环境

Windows 不推荐，PHP程序不推荐Win系统

## 软件环境

### 程序支持环境

- PHP 7.2+
- Mysql 5+
